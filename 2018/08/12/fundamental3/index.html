高数题型（一）----求极限相关题型

发表于 2018-08-12 | 分类于 基础积累
字数统计: 741 | 阅读时长 ≈ 3

摘要

总结一下红宝书里的第一章求极限相关的题型，主要的处理方法有以下几种

  • 1、用初等数学中的恒等变形，约分化简。
  • 2、如果有因式的极限存在但不为0，可将这种因式按乘积运算法则提出来另求。
  • 3、等价无穷小替换
  • 4、洛必达法则
  • 5、佩亚诺余项泰勒公式
  • 6、夹逼准则
  • 7、极限的四则运算定理，复合函数求极限，连续函数求极限，几个重要极限
  • 8、用导数定义求极限

一、如下几种待定型的极限$\cfrac{0}{0} 、0\cdot\infty 、\cfrac{\infty}{\infty}$

  • 1、无穷比无穷型，分子分母为x的多项式或简单根式，提出影响分子分母趋于无穷的因子（同除以$x^n$），可以消去部分次数较低的项。

  • 2、0比0型，幂指函数的常用处理方法是将幂指函数转化成指数函数如

  • 3、0比0型，符合使用洛必达的条件则使用洛必达解决，若分子在x=0处可取任意阶导数，则考虑使用佩亚诺余项泰勒公式展开。展开的阶数与分母相关。

  • 4、0比0型，变限积分表示的函数求极限，采用洛必达法则最合适，对变限积分求导数可以消除积分号。其中可以对变限积分做变量代换（积分结果与符号无关）

  • 5、可以把原式转化为多项式来求解。

  • 1、可将其通分为零比零型，再做处理。

三、如下几种类型求极限$1^{\infty } 、0^{\infty } 、\infty^0$

  • 1、一无穷型将原式凑成重要极限$\lim _{z\rightarrow 0}\left( 1+z\right) ^{\dfrac {1}{z}} = e$

  • 2、可以将原式化成指数函数$U\left( x\right) ^{v\left( x\right) }=e^{v\left( x\right) \ln u\left( x\right) }$

四、求极限相关的证明题

  • 1、含有|x|和$e^{\dfrac {1}{x}}$的x趋于0的极限，含有取整函数[x]的x趋于整数时的极限，含有这些的应分x趋于0正，x趋于0负两种情况分别求解。

  • 2、数列的极限，一般题型为数列n项和或n项积的极限，可以化n项和为积分。或者放大缩小，使用夹逼准则求解。

  • 3、极限运算定理的正确运用

  • 当A和B都存在，则C必都存在，且C = D（C=E）
  • A和B之一存在，另一不存在，则C不存在
  • A和B都不存在，则D和E或者都不存在，或者一个存在，但不可能两个都存在。
  • A和B都存在则F存在且等于AB的积
